# 🌿 PureSkin
 
A responsive skincare landing page showcasing beauty products with a clean and modern design.

## ✨ Features

* Responsive navbar with Bootstrap
* Hero section with image carousel
* Product grid with cards
* Hover effects for better UI
* Footer with links and social media
* Mobile-friendly design (media queries)

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Bootstrap 5

## 🚀 How to Run

1. Download or clone the project
2. Open `index.html` in your browser

## 📌 Notes

This project was built to practice responsive design, Bootstrap components, and UI layout.
